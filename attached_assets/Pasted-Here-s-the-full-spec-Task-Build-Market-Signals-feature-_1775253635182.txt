Signals CRUD:
GET /api/market-signals — fetch all signals for workspace, ordered by created_at DESC. For each signal, also return requirement count:
sqlSELECT ms.*, COUNT(msr.id)::int AS requirement_count
FROM market_signals ms
LEFT JOIN market_signal_requirements msr ON msr.signal_id = ms.id
WHERE ms.workspace_id = ${workspaceId}
GROUP BY ms.id
ORDER BY ms.created_at DESC
POST /api/market-signals — create signal. Body: { name, source_type, source_organisation, signal_date, status, deadline, notes }. Insert into market_signals with workspaceId. Return created row.
PUT /api/market-signals/:id — update signal fields. Verify workspace_id = workspaceId before updating. Return updated row.
DELETE /api/market-signals/:id — delete signal (cascade deletes requirements and links). Verify workspace_id first.
Requirements CRUD:
GET /api/market-signals/:signalId/requirements — fetch all requirements for a signal, with their linked dimension items:
sqlSELECT msr.*,
  COALESCE(
    json_agg(
      json_build_object(
        'id', rdl.id,
        'dimension_id', rdl.dimension_id,
        'dimension_name', rdl.dimension_name,
        'item_name', rdl.item_name
      )
    ) FILTER (WHERE rdl.id IS NOT NULL),
    '[]'
  ) AS linked_items
FROM market_signal_requirements msr
LEFT JOIN requirement_dimension_links rdl ON rdl.requirement_id = msr.id
WHERE msr.signal_id = ${signalId}
AND msr.workspace_id = ${workspaceId}
GROUP BY msr.id
ORDER BY msr.created_at ASC
POST /api/market-signals/:signalId/requirements — create requirement + links in a single transaction. Body: { requirement_text, source_reference, linked_items: [{dimension_id, dimension_name, item_name}] }.
typescript// 1. Insert requirement
const reqResult = await db.execute(sql`
  INSERT INTO market_signal_requirements
    (signal_id, workspace_id, requirement_text, source_reference)
  VALUES
    (${signalId}::uuid, ${workspaceId}, ${requirement_text}, ${source_reference || null})
  RETURNING *
`);
const requirementId = (reqResult.rows[0] as any).id;

// 2. Insert links
for (const item of (linked_items || [])) {
  await db.execute(sql`
    INSERT INTO requirement_dimension_links
      (requirement_id, workspace_id, dimension_id, dimension_name, item_name)
    VALUES
      (${requirementId}::uuid, ${workspaceId}, ${item.dimension_id}::uuid,
       ${item.dimension_name}, ${item.item_name})
  `);
}

// 3. Return requirement with links
const full = await db.execute(sql`
  SELECT msr.*,
    COALESCE(
      json_agg(
        json_build_object(
          'id', rdl.id,
          'dimension_id', rdl.dimension_id,
          'dimension_name', rdl.dimension_name,
          'item_name', rdl.item_name
        )
      ) FILTER (WHERE rdl.id IS NOT NULL),
      '[]'
    ) AS linked_items
  FROM market_signal_requirements msr
  LEFT JOIN requirement_dimension_links rdl ON rdl.requirement_id = msr.id
  WHERE msr.id = ${requirementId}::uuid
  GROUP BY msr.id
`);
return res.json(full.rows[0]);
PUT /api/market-signals/:signalId/requirements/:id — update requirement text + source_reference + replace all links. Delete existing links for this requirement, re-insert new ones. Return updated requirement with links using same query as above.
DELETE /api/market-signals/:signalId/requirements/:id — delete requirement (cascade deletes links). Verify workspace_id first.
Heatmap endpoint:
GET /api/market-signals/heatmap — returns item-level demand counts:
sqlSELECT
  rdl.dimension_id,
  rdl.dimension_name,
  rdl.item_name,
  COUNT(DISTINCT msr.signal_id)::int AS signal_count
FROM requirement_dimension_links rdl
JOIN market_signal_requirements msr ON msr.id = rdl.requirement_id
JOIN market_signals ms ON ms.id = msr.signal_id
WHERE rdl.workspace_id = ${workspaceId}
AND (${statusFilter} = 'all' OR ms.status = ${statusFilter})
GROUP BY rdl.dimension_id, rdl.dimension_name, rdl.item_name
ORDER BY signal_count DESC
Accept optional ?status=active|closed|all query param (default: all).
