Task: Add "Paste text" and "Import from document" tabs to the Add Requirement modal in Market Signals
Do not modify any existing files except server/routes.ts and the new client/src/pages/market-signals.tsx. All extraction logic lives in a new endpoint.

Part 1 — New extraction endpoint in server/routes.ts
POST /api/market-signals/:signalId/extract-requirements
requireAuth only
Accepts either:

{ text: string } — pasted text (max 3,000 chars, enforce server-side)
{ file: base64string, mimeType: 'application/pdf' | 'image/jpeg' | 'image/png' } — uploaded file

Logic:
typescriptconst userId = (req as any).userId;
const wsResult = await pool.query(
  `SELECT id FROM workspaces WHERE user_id = $1 LIMIT 1`, [userId]
);
const workspaceId = wsResult.rows[0]?.id;
if (!workspaceId) return res.status(404).json({ error: 'Workspace not found' });

// 1. Fetch workspace/product context
const prodContext = await storage.getProductContext(workspaceId);
const productName = prodContext?.productName || 'our product';
const industryContext = (prodContext as any)?.industry || 'software';

// 2. Fetch all dimensions + items for this workspace
const dimsResult = await db.execute(sql`
  SELECT id, name, items FROM competitive_dimensions
  WHERE workspace_id = ${workspaceId}
  ORDER BY display_order ASC
`);
const dims = dimsResult.rows as any[];
const dimContext = dims.map((d: any) => {
  const items = (typeof d.items === 'string' ? JSON.parse(d.items) : d.items) || [];
  const itemNames = items.map((i: any) => i.name || i).join(', ');
  return `- ${d.name}: ${itemNames}`;
}).join('\n');

// 3. Build prompt
const systemPrompt = `You are a product intelligence analyst helping extract and categorise requirements.

Context:
- Product: ${productName}
- Industry: ${industryContext}

Available capability dimensions and their items:
${dimContext}

Your job:
1. Read the provided content
2. Extract ONLY genuine capability or feature requirements (ignore legal boilerplate, vendor instructions, deadlines, pricing questions, and administrative text)
3. For each requirement, identify which dimension items it relates to
4. Return results as a JSON array

Return ONLY a JSON array, no markdown, no preamble:
[
  {
    "requirement_text": "The exact requirement as stated in the source",
    "source_reference": "Where it appeared (section, page, question number) or null",
    "linked_items": [
      {
        "dimension_name": "Exact dimension name from the list above",
        "item_name": "Exact item name from that dimension"
      }
    ]
  }
]

Rules:
- requirement_text must be the verbatim or near-verbatim text from the source
- Only link to items that exist exactly in the dimensions list above
- linked_items can be empty array if no clear match
- Extract 3-15 requirements maximum — prioritise specificity over quantity
- Never invent dimension names or item names not in the list above`;

// 4. Build message content
let messageContent: any[];

if (req.body.text) {
  // Enforce 3000 char limit
  const text = (req.body.text as string).substring(0, 3000);
  messageContent = [{ type: 'text', text: `Extract requirements from this text:\n\n${text}` }];
} else if (req.body.file) {
  // File upload — PDF or image
  const { file, mimeType } = req.body;
  if (mimeType === 'application/pdf') {
    messageContent = [
      {
        type: 'document',
        source: { type: 'base64', media_type: 'application/pdf', data: file }
      },
      { type: 'text', text: 'Extract requirements from this document. Focus on capability and feature requirements only, ignore administrative or legal sections.' }
    ];
  } else {
    messageContent = [
      {
        type: 'image',
        source: { type: 'base64', media_type: mimeType, data: file }
      },
      { type: 'text', text: 'Extract requirements from this image. Focus on capability and feature requirements only.' }
    ];
  }
} else {
  return res.status(400).json({ error: 'Either text or file must be provided' });
}

// 5. Call Claude
const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });
let requirements: any[] = [];
try {
  const msg = await anthropic.messages.create({
    model: 'claude-sonnet-4-6',
    max_tokens: 2000,
    system: systemPrompt,
    messages: [{ role: 'user', content: messageContent }]
  });
  const raw = (msg.content[0] as any).text || '[]';
  const clean = raw.replace(/```json|```/g, '').trim();
  const match = clean.match(/\[[\s\S]*/);
  requirements = JSON.parse(match ? match[0] : clean);
} catch (err: any) {
  console.error('[EXTRACT] Claude extraction failed:', err.message);
  return res.status(500).json({ error: 'Extraction failed. Please try again.' });
}

// 6. Enrich with dimension_id by matching dimension_name
const dimMap: Record<string, string> = {};
for (const d of dims) {
  dimMap[d.name] = d.id;
}

const enriched = requirements.map((req: any) => ({
  ...req,
  linked_items: (req.linked_items || []).map((li: any) => ({
    ...li,
    dimension_id: dimMap[li.dimension_name] || null
  })).filter((li: any) => li.dimension_id !== null)
}));

return res.json({ requirements: enriched });
